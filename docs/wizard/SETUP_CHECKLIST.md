# Setup Checklist (10 Etapas)

Abaixo está o detalhamento dos 10 passos do Infrastructure Wizard para acompanhamento offline:

- [ ] **Etapa 1: Ambiente** (Configuração de URLs e ambiente de execução).
- [ ] **Etapa 2: Vercel** (Deploy e vinculação de variáveis de ambiente de produção).
- [ ] **Etapa 3: Supabase** (Postgres, RLS, Storage e triggers).
- [ ] **Etapa 4: GitHub** (Personal Access Token para commit).
- [ ] **Etapa 5: IA** (OpenAI, Gemini e Claude API Keys).
- [ ] **Etapa 6: Google** (OAuth, Analytics 4 e Search Console).
- [ ] **Etapa 7: E-mail** (Domínio e DKIM no Resend).
- [ ] **Etapa 8: Organic Bridge** (Instalação e chaves do publicador nos 5 blogs).
- [ ] **Etapa 9: Publicação** (Aprovação manual, preview e revalidate).
- [ ] **Etapa 10: Go-Live** (Verificação de checklist final).
