# First Install Guide

Este manual descreve o passo a passo para realizar a primeira instalação do Organic Traffic OS com o apoio do assistente interativo.

---

## 1. Iniciando o Processo

1. Acesse o painel central da aplicação e clique no item **Setup Wizard** no menu do sistema (ou acesse diretamente pela rota `/setup`).
2. A tela exibirá seu progresso inicial (provavelmente 0% caso nenhuma chave de produção esteja preenchida).
3. **Siga a ordem sequencial**: comece configurando as variáveis básicas de Ambiente e prossiga passo a passo pelas etapas.
4. Clique no link **Ver Guia de Ajuda** ao final de cada card para ler as instruções específicas de cada provedor externo (Google, OpenAI, Vercel, Supabase).
5. Após preencher as chaves na Vercel e realizar o redeploy, clique em **Revalidar Tudo** no cabeçalho do Wizard para recalcular seu progresso.
